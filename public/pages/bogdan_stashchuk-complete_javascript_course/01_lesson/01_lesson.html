Урок 1: Самое важное в JavaScript

Самое важное в JavaScript это - функции, выражения, объекты

Главная идея в JavaScript в том, что практически все сущности в JavaScript это - объекты. Объект это - набор свойств, имя:значение, либо пара имя:значение. Имена еще называют ключами.

Пример объекта

                
                 // example object

    {
        visible: true,
        colorDepth: 24,
        title: 'My image',
        Orientation: {
            angle: 0,
            type: 'landscape'
        },
        pixelDepth: 24,
        width: 1440
    }
  • Массив это - объект
  • Функция это - объект
  • Число это - объект
  • Строка это - объект

Числа и строки ведут себя как объекты. Значения такие как число и строка это примитивные значения. Объект это набор свойств, и у свойств есть свои значения.

Что такое console.log ?

console.log('Hello world') - это выражение 

                
                 console.log('Hello world')

  • console - это объект. У объекта есть свойства. У каждого свойства есть пара - ключ:значение.
  • log - это метод. Метод это функция, которая является значением одного из свойств объекта.
  • точка между console и log - это синтаксис JavaScript, который называется точечная запись. С помощью точечной записи можно получать доступ к свойствам объектов. В данном случае в console.log('Hello world') мы получаем доступ к методу объекта. Метод называется log
  • Скобки это вызов метода. Это также часть синтаксиса JavaScript. Поставив скобки после log , мы говорим, что мы хотим вызвать эту функцию, которая является методом объекта console.
  • ('Hello world') - Внутри вызова метода находится аргумент, и в данном примере это просто значение типа - строка(string)

метод dir

                
                 console.dir()

Кроме метода log у объекта console есть такой метод как dir И с помощью этого метода можно отобразить в консоли все свойства того или иного объекта. Например с помощью console.dir можно увидеть, что любая функция в JavaScript это объект, и увидеть все свойства этого объекта.

метод table

                
                 console.table()

C помощью метода table можно увидеть в консоли все свойства того или иного объекта, который передан в этот метод как аргумент, который находится внутри скобок в табличном виде. То есть все свойства объекта будут отображены в виде таблицы.

Любое выражение возвращает значение

В консоли браузера во вкладке console наберем команду console.log('Name') Нажимаем enter и получаем следующий результат: В консоли отображается Name, а также виден ответ от интерпретатора JavaScript - undefined
Это результат этого выражения. Метод log возвращает значение undefined Если запустить это выражение в приложении terminal , то результат будет такой же.

console - это объект. Как это проверить? Можно написать console в консоли браузера и нажать enter

Консоль покажет ряд свойств этого объекта. Некоторые из этих свойств имеют функцию как значение, такие свойства называются методами. Эти методы можно вызывать. Найдем метод log , и только что мы вызывали этот метод. Это метод обекта console

Вызов функции это выражение.

Почему это выражение? Потому, что вызов функции всегда возвращает значение. И в данном примере результат вызова функции это значение undefined Это то, что нам вернул метод log

Что такое выражение?

Любое выражение всегда возвращает значение.

Вот некоторые примеры выражений:

  • 'abc' строка
  • 10 число
  • 5 + 2 выражение с использованием оператора +, и результатом такого выражения будет число 7
  • c = 10 Переменной c присваивается число 10 Результатом этого выражения будет то значение, которое присвоено переменной. В данном примере это значение 10
  • 'Good' + 'Evening' Конкатенация строк с помощью оператора + , результатом этого выражения будет одна строка
  • a <= b || c !==d Выражение с несколькими операторами.
  • || оператор "или"
  • !== оператор "не равно"
  • <= оператор "меньше либо равно"
  • myFunction(c,d) Вызов функции. В этом примере вызов функции myFunction с аргументами c и d

Примеры выражений с результатами: 
                
                'abc' // Результат выражения: abc
                
            
                
                 10 // Результат выражения: 10
                
            
                
                 a + b // Результат выражения: сумма 'a' и 'b'
                
            
                
                 'Good ' + 'Evening' // Результат выражения: 'Good Evening' (новая строка)
                
            
                
                 a <= b || c !==d // Результат выражения: true или false
                
            
                
                 myFunction(c,d) // Результат выражения: Результат функции.

                // То, что возвращает функция в конце выполнения.
Выражение присваивания 
                
                 a = 20 // выражение присваивания
Выражения с побочными действиями

Выражение с побочными действиями не только возвращает значение, но и выполняет другие действия.

                
                 a = 5

Кроме возвращения значения, значение 5, приведет к выполнению еще другого действия. Побочное действие - значение 5 будет присвоено перменной a Позже в коде можно будет обратиться к этой переменой и прочесть это значение. То есть были выполнены какие-то действия, кроме возврата значения этого выражения.

Увеличение значения на единицу 
                
                 b++

В этом выражении используется оператор ++ Этот оператор увеличивает значение переменной на единицу. Например, если переменная b содержала цифру 6, то после применения к ней оператора ++ значение этой переменной будет увеличино на единицу и будет равняться цифре 7 Это выражение вернет цифру 7 , и кроме этого значение переменой b увеличится на единицу, то есть будет выполнено другое побочное действие. Побочное действие - увеличение значения переменной b на единицу.

В этом примере мы видим вызов функции 
                
                 myFunction(c,d)

Внутри функции мы можем выполнять различные действия. Внутри функции выполняются другие побочные действия, кроме возврата значения. И так как это выражение, то функция возвращает значение.
